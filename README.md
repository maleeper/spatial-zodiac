# NotebookLM Interview Prep Tutorial

This is a static web application that guides users through using NotebookLM for interview preparation.

## How to Run

Since this is a static application, you do not need to install any dependencies or run a build server.

1.  Simply open the `index.html` file in your web browser.
    -   You can double-click the file in your file explorer.
    -   Or drag and drop the file into a browser window.

## Features

-   **Premium Design**: Modern aesthetics with glassmorphism and smooth animations.
-   **Responsive**: Works on desktop and mobile.
-   **Dynamic Content**: Steps are rendered from a data file, making it easy to update.

## Tech Stack

-   HTML5
-   CSS3 (Vanilla with Variables)
-   JavaScript (ES6+)
-   Lucide Icons (via CDN)
-   Google Fonts (Outfit)
